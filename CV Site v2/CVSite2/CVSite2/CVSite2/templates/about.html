Ik ben Tom Saccasyn, en 3 de jaars student Netwerk & Security aan de Erasmus Hogeschool Brussel.


Ik maak deze site voor ITalent3. Deze site heeft als bedoeling om aan te tonen dat ik een site kan maken met Flask en Python.


Hieronder volgt de uitleg over hoe de api moet worden aangesproken.

De api kan worden aangesproken eenmaal de site online is via postman of door in de console naar de folder te gaan waar het runserver.py bestand in zit en daar het commandprompt te openen.
Daarna moet er in de commandprompt curl worden getyped en dan de gewenste URL waar de data van moet worden gehaald.

Om de api aan te spreken moet men eerst deze site online hebben en dan kan men de data krijgen via een GET request door volgende links te gebruiken:

http://localhost:5555/api/all
http://localhost:5555/api/overmij
http://localhost:5555/api/historiekscholen
http://localhost:5555/api/talenkennis
http://localhost:5555/api/computervaardigheden
http://localhost:5555/api/hobbies
